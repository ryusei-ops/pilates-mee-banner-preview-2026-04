## プレビュー

GitHub Pages で `banner-preview.html` を開くとブラウザで閲覧できます。

## 対象店舗

守谷 / 柏 / 阿佐ヶ谷 / 巣鴨 / 日吉 / 相模大野 / 神戸元町 / 青物横丁 / 笹塚 / 茨木 / 大森

## 構成

- **パターンA**: ビジュアル中心 / コンパクト訴求
  - エリアフック + 最大3人の少人数制 + サブコピー + 体験料0円/入会金0円
- **パターンB**: テキストベース / 手紙風(写真散りばめ)
  - 駅周辺呼びかけ + ストーリー本文 + 先着20名限定バッジ

## フォーマット

各店舗 × 各パターン:
- 正方形 1080×1080 (Meta フィード)
- 縦長 1080×1920 (Meta ストーリー/リール)

## 特別仕様

- **守谷**: 駐車場訴求追加（ロードサイド型、アクロスモール守谷近接）
- **巣鴨**: 40〜50代寄せ、メインコピー大きめ、長年の肩こり訴求
- **神戸元町**: JR元町駅サブフック併記
